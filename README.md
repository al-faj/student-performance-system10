## Quick Start

### Run Locally

**Backend:**
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

**Frontend:**
```bash
cd frontend
npm install
npm start
```

### Deploy on Render

1. Push to GitHub
2. Go to render.com
3. Connect repository
4. Deploy automatically
